# EZ Capture Page

EZ Capture Page is a web-based capture page and lead management system designed for affiliate marketers and small businesses. The platform allows users to create conversion-focused pages, collect leads, and route submissions through a structured backend workflow.

## Tech Stack
- PHP
- MySQL (PDO)
- HTML5 / CSS
- JavaScript
- Bootstrap

## Key Features
- Dynamic capture page generation
- Secure form handling and lead storage
- Modular PHP architecture
- Webhook processing support
- Responsive, mobile-friendly layouts

## Architecture Notes
- Configuration values are centralized
- Sensitive credentials are intentionally excluded from version control
- Designed for deployment on shared or VPS hosting environments

## Setup (Local)
1. Clone the repository
2. Configure database credentials locally (not included in repo)
3. Point web server to the project root
4. Access via browser

## Disclaimer
This repository represents a production-style system with sensitive data intentionally removed for security.
